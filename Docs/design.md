# Design Document

**Project:** Rarebird Infra Website
**Version:** 1.0 (Reference-Based Design System)
**Prepared by:** Archit Mishra

---

## 1. Design Objective

The design aims to create a **premium, modern real estate brand presence** that reflects:

* Scale and credibility
* High-value developments
* Clean, editorial-style presentation

The visual direction is derived from the provided reference and adapted for **township-focused storytelling**.

---

## 2. Design Principles

* Minimal and uncluttered layout
* Strong emphasis on typography
* Image-driven storytelling
* Clear visual hierarchy
* Consistent spacing and alignment

---

## 3. Color System

| Usage          | Color            |
| -------------- | ---------------- |
| Background     | #F5F5F5          |
| Primary Text   | #242424          |
| Secondary Text | Muted grey tones |
| Surface        | #FFFFFF          |

No strong accent colors are used in this phase.

---

## 4. Typography

### Primary Typeface

* Serif font (e.g., Instrument Serif or equivalent)

**Usage:**

* Hero headings
* Section titles

---

### Secondary Typeface

* Sans-serif font

**Usage:**

* Body text
* UI elements
* Labels

---

### Typography Characteristics

* Large-scale headings
* Combination of regular and italic serif styles
* Generous line spacing

---

## 5. Layout System

### Structure

* Centered container layout
* Maximum width with consistent margins

### Spacing

* Large vertical spacing between sections
* Clear separation of content blocks

---

## 6. Component Design

---

### 6.1 Navigation Bar

* Positioned at top
* Minimal layout
* Transparent over hero section

---

### 6.2 Hero Section

* Full-screen background image
* Text overlay (left-aligned)
* Large serif headline
* Subtle overlay for readability

---

### 6.3 About Section

* Split layout:

  * Heading on left
  * Content on right

---

### 6.4 Statistics Section

* Circular or soft-shaped containers
* Large numeric values
* Minimal labels

---

### 6.5 Card Components

Used for:

* Services
* Townships

**Characteristics:**

* Image-first layout
* Rounded corners
* Clean typography
* Subtle hover interaction

---

### 6.6 Image Showcase Section

* Full-width image
* Minimal overlays
* Focus on visual impact

---

### 6.7 Blog Section

* Editorial layout
* Text-focused presentation
* Minimal use of imagery

---

### 6.8 Form Design

* Minimal input fields
* Clean borders
* Consistent spacing
* Prominent submit button

---

### 6.9 Footer

* Simple layout
* Clear hierarchy of links
* Neutral styling

---

## 7. Interaction Guidelines

* Smooth transitions (200–300ms)
* Subtle hover effects
* No heavy animations

---

## 8. Imagery Guidelines

Use:

* Township aerial views
* Architectural photography
* Clean, high-resolution visuals

Avoid:

* Cluttered compositions
* Low-quality assets

---

## 9. Adaptation from Reference

| Reference Element      | Adaptation                        |
| ---------------------- | --------------------------------- |
| Property search bar    | Removed                           |
| Listing marketplace UI | Removed                           |
| Property grid          | Replaced with township showcase   |

---

## 10. Responsive Behavior

### Mobile

* Single-column layout
* Collapsible navigation

### Tablet

* Two-column grid where applicable

### Desktop

* Full multi-column layout

---

## 11. Design Consistency Requirements

* Consistent spacing across sections
* Unified typography scale
* Reusable card and layout patterns
* Visual balance between text and imagery
